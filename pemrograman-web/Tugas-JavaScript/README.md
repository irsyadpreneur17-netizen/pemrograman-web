# Tugas JavaScript Dasar

## Deskripsi Program

Program ini dibuat menggunakan HTML, CSS, dan JavaScript untuk menerapkan konsep dasar JavaScript, manipulasi DOM, event handling, dan validasi form.

## Fitur Program

### 1. Ganti Warna Background
Program memiliki 3 tombol warna:
- Putih
- Biru
- Kuning

Saat tombol diklik, background halaman akan berubah sesuai warna yang dipilih menggunakan manipulasi DOM dan event click.

### 2. Tampilkan Nama Mahasiswa
Pengguna dapat memasukkan nama pada input form. Setelah tombol ditekan, program akan menampilkan sapaan:

Halo, [nama]!

Program menggunakan DOM manipulation dan event handling.

### 3. Event Keyup
Saat pengguna mengetik nama pada input, teks akan otomatis berubah menjadi:

Anda mengetik: ...

Fitur ini menggunakan event keyboard `keyup`.

### 4. Validasi Form Login
Program memiliki validasi form login dengan ketentuan:
- Username tidak boleh kosong
- Password minimal 8 karakter

Jika data tidak valid, pesan error akan tampil di bawah input.
Jika valid, program menampilkan pesan:

Login berhasil!
